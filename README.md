## About

A clone of Discord with features like realtime chat app.
- React is used along with Redux to power front-end.
- React Routing with Error handling 
- Firebase is used for authentication and firestore to store messages, users.
- Material UI is used along with CSS with a little of styled components for UI.


## Features
- Real time messaging b/w multiple users.
- Load user data upon login.
- Firebase Authentication using google.

## Installation
```js
#install dependencies
$ npm install

build for production and launch server
$ npm run build
$ npm run start
```
